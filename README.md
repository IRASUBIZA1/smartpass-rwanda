# SmartPass Rwanda 🇷🇼

### Learn • Practice • Succeed

SmartPass Rwanda is an educational platform dedicated to supporting Rwandan students with quality learning resources. The platform provides access to past papers, study notes, books, and practice materials to help learners prepare effectively for national examinations and achieve academic success.

## 🌐 Live Website

https://irasubiza1.github.io/smartpass-rwanda/

## 📚 Features

- Past Papers Library
- Books Library
- Study Notes
- Practice Quizzes
- Exam Preparation Resources
- Candidate Support Materials

## 🎯 Mission

To make quality educational resources accessible to every student in Rwanda and promote academic excellence through technology.

## 🚀 Vision

To become Rwanda's leading online learning platform for secondary school students and national examination candidates.
