# SmartPass Rwanda 🇷🇼

Learn • Practice • Succeed

SmartPass Rwanda is an educational platform designed to help Rwandan students access past papers, books, notes, and quizzes for better exam preparation.

## 🌐 Live Website

https://irasubiza1.github.io/smartpass-rwanda/

## 📚 Features

- Past Papers
- Books Library
- Study Notes
- Quizzes
- Exam Preparation Resources

## 🎯 Mission

To help every Rwandan student learn, practice, and succeed.
